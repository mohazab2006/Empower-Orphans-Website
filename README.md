# 🌍 Empower Orphans Website

A full-stack web platform built for a nonprofit organization to support donations, event management, and chapter coordination, while enhancing donor engagement through a modern and accessible interface.

---

## 🚀 Overview

This project is a scalable web application designed to:

- Support nonprofit operations including donations, events, and chapter management  
- Provide a responsive and accessible user experience across all devices  
- Enable structured user roles and permissions  
- Enhance donor interaction and engagement  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS  
- **Backend & Database:** Supabase (PostgreSQL)  
- **Infrastructure:** Docker, NGINX, SSL/TLS  

---

## ✨ Features

### Core Functionality

- 💻 Full-stack web application for nonprofit operations  
- 📱 Fully responsive UI with strong accessibility support  
- 🧩 Modular and scalable architecture  

### 🔐 Role-Based Access Control (RBAC)

- Admin, chapter, and volunteer role management  
- Secure access to protected resources  
- Structured permission system for different user types  

### 🤝 Donor Interaction

- Donation-ready system architecture  
- Interfaces designed to improve donor engagement  
- Clear and intuitive user flows for interacting with the platform  

---

## ⚙️ Deployment

- Deployed on a self-managed VPS  
- Containerized using Docker  
- Reverse proxied with NGINX  
- Secured with SSL/TLS for production  

---

## 📌 Key Highlights

- Built a production-ready full-stack application  
- Implemented secure role-based access control (RBAC)  
- Designed for real-world nonprofit use cases  
- Focused on performance, scalability, and user experience
